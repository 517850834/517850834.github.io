JavaScript数值

发表于 2016-07-08   |   分类于 JavaScript   |  

在JavaScript中数值的构造函数是Number函数，最常用的方法是直接使用字面亮来生成一个数值，也可以使用Number函数生成一个数值包装对象。

相关属性

Number.MAX_VALUE

返回JavaScript中能表示的最大值1.7976931348623157e+308。

Number.MIN_VALUE

返回JavaScript中能表示的最小值5e-324。

NaN

表示这个数值不是一个数值，NaN与自身不相等。

实例方法

toFixed

toFixed方法通过四舍五入将数值的小数位保留到指定的位数，然后返回该数值的字符串形式。

实例

toString

toString方法返回数值的字符串形式，如果不传入参数则默认以10进制转换为字符串。

参数
转换成字符串的基数，例如二进制、八进制、16进制等。

实例

全局方法

isNaN

isNaN方法用来判断指定数值是否为NaN。

实例

parseInt

parseInt方法将字符串以指定的进制转换成10进制整数。

参数一
指定需要转换成整数的字符串

参数二
指定字符串的进制，默认为10进制。

实例

parseFloat

parseFloat方法将字符串转换成浮点数值。

实例

Math对象

Math.abs

Math.abs方法返回指定数值的绝对值。

实例

Math.ceil

Math.ceil方法返回一个向上取整的数值。

实例

Math.floor

Math.floor方法返回一个向下取整的数值。

实例

Math.max

Math.max方法返回一组数值中最大的数值。

实例

Math.min

Math.min方法返回一组数值中最小的数值。

实例

Math.pow

Math.pow方法返回指定数值的指定次幂的数值。

参数一
指定数值基数。

参数二
指定次幂。

实例

Math.random

Math.random方法返回一个大于等于0，小于1的随机数值。

实例

Math.round

Math.round方法返回一个四舍五入后的整数数值。

实例

Math.PI

Math.PI属性返回圆周率。
