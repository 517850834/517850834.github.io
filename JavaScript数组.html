JavaScript数组

发表于 2016-07-08   |   分类于 JavaScript   |  

属性

length

length属性返一般情况下返回的是数组中元素的个数，但在不连续数组中则返回的是最大索引值+1。

实例

常用方法

添加删除修改排序

以下方法会直接修改原数组。

push

push方法在数组最后一位添加新的元素，函数的返回值为最新的length属性值。

实例

pop

pop方法会把数组中最后最后一位元素删除，函数的返回值为删除的数组元素。

实例

unshift

unshift方法在数组的第一位添加新的元素，函数的返回值为最新的length属性值。

实例

shift

shift方法会把数组中的第一位元素删除，函数的返回值为删除的数组元素。

实例

splice

splice方法将数组中指定索引值开始的元素删除然后替换成新的元素，函数的返回值为删除的数组元素组成的数组，如果没有删除则返回空数组。

参数一
需要删除修改元素的开始索引

参数二
需要删除元素的个数

余下参数
添加进入数组的新元素

实例

sort

sort方法默认将数组中的元素按照Unicode码位点进行进行排序。

可选参数
传入一个指定按照某种顺序进行排序的函数。

实例

reverse

reverse方法将数组中元素的位置进行颠倒。

实例

复制拼接索引

slice

slice方法将数组中指定索引位置的元素浅复制到一个新的数组，如果没有传入参数则默认复制整个数组，参数为负数则表示倒数。

参数一
需要复制的数组元素的开始索引。

参数二
需要复制的数组元素的结束索引，复制的结果不包含结束索引位置的元素，如果不传入则默认复制到最后一个数组元素。

实例

concat

concat将传入的数据与原数组合并成新的数组返回。

实例

join

join方法将所有数组元素合并成一个新的字符串返回，不传入指定分隔符则每个数组元素之间使用,号连接。

实例

indexOf

indexOf方法在数组中从做往右搜索指定元素，返回第一个符合的元素的索引位置，如果不存在则返回-1。

参数一
需要查询索引的元素。

参数二
从指定索引值开始查询。

实例

IE9以下不支持

lastIndexOf

lastIndexOf方法与indexOf方法一样，区别在于是从右往左搜索。

IE9以下不支持

遍历

数组的遍历方法可以接受一个处理函数以及一个对象来当做处理函数中的this值这两个参数，处理函数一般接受三个参数，分别表示当前数组元素，当前索引，数组本身。

例子

forEach

forEach方法让数组中的每一项都执行一次传入的函数，传入的函数不需要返回值。

实例

IE9以下不支持

map

map方法返回一个原数组中的每个元素调用一个指定函数后返回的值组成的一个新的数组。

实例

IE9以下不支持

every

every方法用来检测数组元素的每一项是否全部符合函数中指定的测试条件，返回布尔值。

实例

IE9以下不支持

some

some方法用来检测数组元素的每一项是否部分符合函数中指定的测试条件，返回布尔值。

实例

IE9以下不支持

filter

filter方法返回符合指定函数测试条件的数组元素组成的新的数组。

实例

IE9以下不支持

reduce

reduce方法接受一个函数作为累加器，把数组中的每个元素从左往右合并成为一个值。

实例

IE9以下不支持
