JavaScript数值

发表于 2016-07-08   |   分类于 JavaScript   |  

在JavaScript中数值的构造函数是Number函数，最常用的方法是直接使用字面亮来生成一个数值，也可以使用Number函数生成一个数值包装对象。

相关属性

Number.MAX_VALUE

返回JavaScript中能表示的最大值1.7976931348623157e+308。

Number.MIN_VALUE

返回JavaScript中能表示的最小值5e-324。

NaN

表示这个数值不是一个数值，NaN与自身不相等。

实例方法

toFixed

toFixed方法通过四舍五入将数值的小数位保留到指定的位数。

实例

toString

isNaN方法用来判断指定数值是否为NaN。

参数一

参数二

parseFloat方法将字符串转换成浮点数值。

JavaScript字符串

发表于 2016-07-07   |   分类于 JavaScript   |  

在JavaScript中字符串的构造函数是String函数，最常用的方法是直接字面亮写法来生成一个字符串，也可以使用String函数生成一个字符串包装对象。

属性

length

‘length’属性是字符串对象的的唯一属性，纪录了指定字符串的长度。

常用方法

由于字符串是基本类型数据，所以下面操作字符串的方法全部都是返回一个新的数据，原字符串数据不会有任何改变。

indexOf

indexOf方法用来从左往右获取字符串中指定字符的索引位置，如果不存在则返回-1。

参数一
需要获取索引位置的指定字符。

参数二
指定从某个索引位置开始往右查询。

实例

lastIndexOf

lastIndexOf方法用来从右往左获取字符串中指定字符的索引位置，如果不存在则返回-1。

参数一
需要获取索引位置的指定字符。

参数二
指定从某个索引位置开始往左查询。

实例

charAt

charAt方法用来获取字符串中指定索引位置的字符，如果不存在则返回空字符串。

实例

slice

slice方法用来从左往右的顺序提取字符串中指定的一部分字符，如果结束索引值小于开始索引值，则返回空字符串。

参数一
需要提取字符的开始索引值，如果不填写结束索引值参数则默认为提取到最后一位，索引值可以为负数，表示倒数。

参数二
需要提取字符的结束索引值，提取的结果不包含结束索引值位置的字符。

实例

substring

substring方法与slice方法基本上一样，但传入的索引值为负数或者NaN则被当作0，传入的结束索引值如果小于开始索引值，则自动替换开始与结束索引值的位置。

实例

substr

substr用来提取字符串中指定的一部分字符。

参数一
需要提取字符的开始索引值，如果不填写提取字符的数量参数，则默认提取到最后一位，如果参数为负数则表示倒数。

参数二
需要提取字符的数量，如果为负数则表示0返回空字符串。

实例

match

match方法用来提取符合正则表达式规则的字符，返回一个包含该字符元素的数组。如果传入的不是正则表达式则被隐式的转换为正则表达式。

实例

relpace

replace方法用来将字符串中符合正则表达式规则的字符替换成一个新的字符。

参数一
接受一个正则表达式或者字符

参数二
符合匹配到的字符被替换的新字符或者一个函数

实例

concat

concat方法将一个或者多个数据与原字符串连接合并成新的字符串(如果传入的参数不为字符串则会被隐式的转换为字符串)。

实例

split

split方法将一个字符串分割成为一个数组。

参数一
指定分割字符串的字符，如果没传入这个参数则返回整个字符串作为数组的一个元素。

参数二
限定返回分割成数组的字符串元素数量。

实例

trim

trim方法用来将字符串左右两端的空白清除，返回一个新的字符串。

实例

toLowerCase

toLowerCase方法将字符串全部转换成小写。

实例

toUpperCase

toUpperCase方法将字符串全部转换成大写。

JavaScript类型转换

发表于 2016-07-06   |   分类于 JavaScript   |  

在JavaScript中发生数据类型转换的场景分为人为主动进行类型转换与在一些运算符与语句中隐式类型转换。

主动类型转换

转换为数值

可以使用Number函数将任何类型的数据转换成数值。

转换为字符串

使用Srting函数可以将任何类型的数据转换为字符串。

转换为布尔值

Boolean方法可以将任何类型数据转换为布尔值。

隐式类型转换

默认转换为布尔值

在if、while语句、取非操作符、三元操作符的条件的部分都会使用Boolean函数自动转换为布尔值。

默认转换为字符串

在字符串与其它类型数据使用+号操作符进行运算时会使用String函数将其它类型数据转换成字符串。

自动转换为数值

除了+号运算符遇到字符串以外，其它的算数运算符在需要类型转换时都将数据使用Number函数转换成数值。

JavaScript类型判断

发表于 2016-07-06   |   分类于 JavaScript   |  

目前知道的JavaScript中类型判断有以下四种方法，其中使用constructor属性这种方法稍微稍微加工下就可以判断出所有的数据类型，而其他的方法都不能做到。

typeof

typeof操作符在一般情况下可以判断出undefined、boolean、number、string、function这几种数据类型，其他的类型都会被认定为object类型。

但是在使用对象创建法创建基本类型数据的情况下，typeof操作符只能判断出undefined与function这两种数据类型，其他的都会被认定为object类型。

instanceof

instanceof操作符用来判断指定对象原型链上的constructor属性是否为指定的函数(基本上很少使用)，如果是则返回true，否则返回false。只能用来判断JavaScript的内置对象与自定义对象，用来判断基本数据类型则全部返回false(如果使用对象创建法创建的基本类型数据则可以来判断)。

Object.prototype.toString

Object.prototype.toString这个方法可以将对象转为其字符串表示法，可以用来判断基本数据类型与内置对象，但是自定义对象会被判断为object。

constructor

constructor属性指向的是这个对象的构造函数，除了undefined与null这两种数据类型不能使用以外，其他的所有数据类型都可以使用。因为在JavaScript中，除了undefined与null以外，其他所有的数据类型都是对象，而对象都是有其构造函数的，constructor这个属性就指向了该对象的构造函数。
