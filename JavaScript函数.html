JavaScript函数

发表于 2016-07-09   |   分类于 JavaScript   |  

属性

length

length属性指明函数形参的个数。

实例

方法

apply

apply方法用来指定函数的this值，传入给函数的参数以数组的形式传入。

参数一
函数内部this值指向的对象。

参数二
以数组的形势传入的参数。

实例

call

call方法用来指定函数的this值，传入给函数的参数以普通单个的形式传入。

参数一
函数内部this值指向的对象。

参数二
以普通形式传入的参数。

实例

bind

bind方法与call方法基本上是一样的，只是返回一个函数来等到需要的时候再运行。

参数一
返回函数内部的this值指向的对象。

参数二
以普通形式传入的参数。

实例

IE9以下不支持

arguments对象

arguments对象是存在于函数内部的一个类数组对象，可以使用访问数组内部元素的方式来访问arguments对象中传入给函数的参数，但不会继承数组的方法。

arguments.length

arguments.length属性值返回实际传入给函数的参数个数。

arguments.callee

arguments.callee属性返回的是函数自身。

实例

构造函数

函数可以当做构造函数来使用new操作符创建新的对象，构造函数的函数名称通用是使用大写，函数默认返回一个对象。

实例

prototype属性

构造函数的prototype属性用来给构造函数所产生出来的所有对象提供一些通用的属性和方法。

实例

this值

this值是存在于函数内部的一个关键字，指向的是函数执行的上下文环境)。

作为对象的方法使用

函数在作为对象的方法被启动时，this值指向的就是那个对象。

实例

作为构造函数使用

函数在作为构造函数启动时，this值指向的是构造函数返回的的对象。

实例

使用apply/call/bind方法

在函数上使用apply、call、bind方法可以来指定函数内部的this值所指定的对象。

实例

全局环境与函数中使用

函数在全局环境中直接执行，内部的this指向的是window这个全局对象(在严格模式下会报错)。

实例

在函数中执行函数，这时被执行的函数的this值指向的也是window全局对象(严格模式下会报错)。

实例

作为DOM事件处理函数使用

函数被作为DOM事件处理函数来使用时，this值指向的是发生事件的DOM节点。

实例

作为内联事件处理函数使用

函数直接在HTML元素中作为内联事件处理函数来执行，this值指向的是函数所在的DOM节点。

实例

函数递归

函数递归就是在函数内部执行自身函数来达到循环的效果，需要有一个退出递归的条件。

实例

函数柯里化

函数柯里化是一个函数接受依次接受一些参数，但是不会立即返回结果，等到需要返回结果的时候再开始求值返回结果。
